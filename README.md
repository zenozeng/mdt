# mdt

Markdown toolkit (pandoc, XeLaTeX, mermaid and LaTeX beamer slide show)

## Usage

- Build docker image

  ```bash
  mdt docker
  ```

- Convert markdown to pdf

  ```bash
  cat examples/beamer.md | mdt pdf
  ```
